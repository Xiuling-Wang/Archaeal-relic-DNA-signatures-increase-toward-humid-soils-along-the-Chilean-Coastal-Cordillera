# Variables and abbreviations

## DNA pools

- `iDNA`: intracellular DNA; interpreted as a cell-associated, putatively active fraction.
- `eDNA`: extracellular DNA; interpreted as an extracellular or relic fraction.

## Sites

- `AZ`: Pan de Azúcar
- `SG`: Santa Gracia
- `LC`: La Campana
- `NA` / `NB`: Nahuelbuta. Some raw metadata use `NB` to avoid confusion with missing values; manuscript text uses `NA`.

## Soil depths

Depths are represented as midpoint values for statistical analyses:

- 0-5 cm -> 2.5 cm
- 5-10 cm -> 7.5 cm
- 10-20 cm -> 15 cm
- 20-40 cm -> 30 cm
- 40-60 cm -> 50 cm

## Environmental variables

The final dbRDA model uses variables retained after VIF screening:

- pH
- conductivity
- moisture
- CN
- Feo
- Alo
- Mno
- Sio
- NH4
- NO3
- Pi
- Po
